## 🧠 Overview

This repository contains all major programming projects completed in **CS 271: Data Structures** using C++. Each project emphasizes a different core structure or algorithm, including linked lists, trees, graphs, hash tables, and sorting. All code is written using modular, class-based C++ and tested through custom drivers and makefiles.

---

## 📂 Project List

### 🔹 **Project 0 – Doubly Linked List**
- Implements a basic doubly linked list with full memory management
- Includes constructor, deep copy, destructor, and operator overloads
- [View README »](./project0-doublylinkedlist/README.md)

---

### 🔹 **Project 1 – Sorting Algorithms**
- Insertion Sort, Merge Sort, Quick Sort, Improved Quick Sort, and Randomized Quick Sort
- Tested for runtime performance on both random and descending arrays
- Includes CLI test driver and Makefile
- [View README »](./project1-sorting/README.md)

---

### 🔹 **Project 2 – Binary Search Tree & Red-Black Tree**
- Implements both BST and RBT with full insert/delete/rebalance logic
- Includes property testing, fix-up methods, and uniform documentation
- [View README »](./project2-trees/README.md)

---

### 🔹 **Project 3 – Hash Tables**
- Implements custom `HashMap`, `HashSet`, and `HashMapTree` structures
- Supports chaining, custom hashing, and performance comparisons
- Includes custom exceptions and vector-backed Set class
- [View README »](./project3-hashtables/README.md)

---

### 🔹 **Project 4 – Graphs (BFS/DFS)**
- Implements unweighted graphs with adjacency lists
- Includes BFS, DFS with visit tracking, and exception handling
- [View README »](./project4-graphs/README.md)

---

### 🔹 **Final Project – Dijkstra’s Algorithm on Weighted Graphs**
- CLI-based pathfinding app using real-world campus map data
- Implements Dijkstra’s algorithm, priority queue, and weighted adjacency list
- Clean CLI interface + interactive testing modes
- [View README »](./final-dijkstra/README.md)
